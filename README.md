# Caffeine Curve — Marketing Site

This is the source code for the [Caffeine Curve](https://delalic.github.io/caffeine-curve/) marketing website.

## Live Site
Visit the live site here: **[https://delalic.github.io/caffeine-curve/](https://delalic.github.io/caffeine-curve/)**

## About
Caffeine Curve is a simple, no-build-step static site consisting of HTML, CSS, and minimal vanilla JavaScript.

## Files
- `index.html` — landing page (hero, features, showcase, privacy callout, what's new)
- `releases.html` — full version history
- `privacy.html` — privacy policy
- `style.css` — all styles (uses Caffeine Curve design tokens)
- `app.js` — scroll reveal + hero parallax
- `assets/` — logos and iPhone/Watch screenshots
